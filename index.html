blackfork

The idea behind this program was to write a genetic algorithm for playing blackjack. I wrote this in conjunction with a friend of mine during our Artificial Intelligence class at St. Paul's School. We began the project by outlining exactly what kinds of containers we wanted for the individual parts of the game—cards, deck(s), hands, etc. In our implementation, the functions only modify these specified containers, which prevented spaghettification of our code.

Further, we developed an algorithm which would rely on parameters derived from the statistics about the game (number and value of cards left in the pile, estimating potential values for the dealer's hand) and randomize its playstyle in relation to those parameters. After many games with this randomization it would select the best outcome and thus best values as the final result.
